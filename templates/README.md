ElCamino.AspNetCore.Identity.CosmosDB.Templates
==================

This package provides the ASP.NET Identity Core project Razor and MVC templates for the ElCamino.AspNetCore.Identity.CosmosDB package that using Azure CosmosDB with the native SQL API instead of the Entity Framework / MSSQL data provider.
